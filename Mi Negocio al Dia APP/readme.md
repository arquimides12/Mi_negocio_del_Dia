# FRONTEND - MI NEGOCIO AL DIA (APP)

1. REQUISITOS:
   - Node.js instalado.
   - Aplicacion "Expo Go" en el celular.

2. INSTALACION DE LIBRERIAS (Copiar y pegar en terminal):
   npm install

3. CONEXION CON EL BACKEND:
   - La App busca al servidor en: http://192.168.1.18:3000
   - Si el backend cambia de IP, debes actualizarla en las pantallas:
     * Inventario.jsx
     * Reportes.jsx
     * Caja.jsx
     * ProductoForm.jsx

4. EJECUCION:
   npx expo start

5. USO:
   - Escanea el codigo QR desde la app Expo Go en tu celular.
   - Asegurate de que el celular y la PC esten en la misma red Wi-Fi.
